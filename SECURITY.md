# Security

## Supported versions

Security fixes are applied to the default branch of this repository (`main`). Use the latest commit for development.

## Reporting a vulnerability

**Do not** open a public GitHub issue for security vulnerabilities.

Email **security@inquiroresearchlabs.ai** with:

- Description of the issue and impact
- Steps to reproduce (proof-of-concept, if possible)
- Affected component (mock server, verifier, CI, examples, etc.)

We aim to acknowledge receipt within a few business days. This inbox is for security reports only.

## Scope

This repository is a **public SDK**: mock MCP server, sample payloads, client examples, and offline verifiers. It does **not** include the production RNA ledger service. Reports about the hosted API should include enough detail for us to reproduce against the documented public endpoint without abusive load.
